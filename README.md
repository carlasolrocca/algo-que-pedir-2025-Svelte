# Algo que pedir Frontend 2025 (Svelte) - Algoritmos III
[![Build](https://github.com/algo3-unsam/algo-que-pedir-2025-grupo2/actions/workflows/build.yml/badge.svg?branch=dev)](https://github.com/algo3-unsam/algo-que-pedir-2025-grupo2/actions/workflows/build.yml)

Trabajo integrador de la materia Algoritmos III donde se trabajo usando Svelte para el desarrollo del front de la aplicación comenzada en Algoritmos II. En esta entrega se diseñó el lado del Administrador de la aplicación, el cual editaba los locales, sus ingredientes y platos disponibles, y gestionaba los pedidos.

### Cómo ejecutar
 
1. Clonar el repositorio
2. Clonar el repositorio del <a href="https://github.com/carlasolrocca/algo-que-pedir-backend-2025-Algo3">backend</a> y levantarlo en el puerto 8080
3. Instalar dependencias:
   ```bash
   yarn install
   ```
4. Levantar la app con
   ```bash
   yarn run dev
   ```
